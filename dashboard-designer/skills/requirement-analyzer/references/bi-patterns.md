# BI看板设计模式参考

本文档定义了BI驾驶舱和看板的常用设计模式。

## 布局模式

### 1. 概览页布局

```
┌─────────────────────────────────────────────────────────────┐
│  顶部导航栏 (标题 + 全局筛选器 + 操作按钮)                    │
├─────────────────────────────────────────────────────────────┤
│  KPI卡片区 (4-6个关键指标卡，横向排列)                       │
├─────────────────────────────────────────────────────────────┤
│  主图表区 (左: 趋势图 60% | 右: 占比图 40%)                  │
├─────────────────────────────────────────────────────────────┤
│  明细表格区 (数据明细表格 + 分页)                            │
└─────────────────────────────────────────────────────────────┘
```

### 2. 分析页布局

```
┌────────────┬────────────────────────────────────────────────┐
│  筛选面板  │  主内容区                                      │
│  (左侧)    │  ┌──────────────────────────────────────────┐  │
│            │  │ 图表区域                                 │  │
│  - 时间    │  └──────────────────────────────────────────┘  │
│  - 区域    │  ┌──────────────────────────────────────────┐  │
│  - 产品    │  │ 数据表格                                 │  │
│  - 指标    │  └──────────────────────────────────────────┘  │
└────────────┴────────────────────────────────────────────────┘
```

### 3. 大屏布局

```
┌─────────────────────────────────────────────────────────────┐
│  标题栏 (居中大标题)                                         │
├──────────────┬──────────────────────────┬──────────────────┤
│  左侧面板    │  中央主区域              │  右侧面板        │
│  (KPI+图表)  │  (地图/核心图表)         │  (KPI+图表)      │
│              │                          │                  │
│              │                          │                  │
└──────────────┴──────────────────────────┴──────────────────┘
```

## 组件类型

### KPI指标卡

```
┌─────────────────────┐
│  📊 销售额          │  <- 图标 + 指标名
│  ¥ 1,285,000       │  <- 主数值
│  ↑ 12.3% 环比      │  <- 变化趋势
│  目标完成: 85%     │  <- 辅助信息
└─────────────────────┘
```

**设计要点**：
- 主数值突出显示
- 趋势使用颜色区分（红绿）
- 可选：进度条、迷你趋势图

### 趋势图

- 折线图：展示指标随时间变化
- 面积图：强调趋势和量级
- 柱状图：对比不同时间点

### 占比图

- 饼图：展示部分与整体关系
- 环形图：饼图变体，更现代
- 矩形树图：多层级占比展示

### 表格

- 基础表格：数据明细展示
- 汇总表格：带小计和合计
- 交叉表格：多维度交叉分析

### 地图

- 区域地图：按区域展示数据分布
- 热力地图：展示密度分布
- 标记地图：展示具体点位

## 交互模式

### 筛选联动

```
用户选择筛选条件 → 所有相关组件刷新数据
```

### 下钻分析

```
点击图表元素 → 展开更详细的数据层级
例: 点击"华东" → 展示华东各省份明细
```

### 联动高亮

```
鼠标悬停图表元素 → 其他相关图表高亮对应数据
```

### 钻取跳转

```
点击组件 → 跳转到详情页面
例: 点击KPI卡 → 跳转到该指标的分析页
```

## 配色建议

### 科技蓝风格

- 背景: #0a1628 (深蓝黑)
- 主色: #00d4ff (科技蓝)
- 强调: #ff6b35 (橙色)
- 文字: #ffffff (白色)

### 简约素雅风格

- 背景: #f8fafc (浅灰白)
- 主色: #4f46e5 (靛蓝)
- 强调: #f59e0b (琥珀)
- 文字: #1e293b (深灰)
