# ASCII线框图模板

本文档提供各种BI看板组件的ASCII线框图模板。

## 页面整体布局

### 概览页布局

```
┌─────────────────────────────────────────────────────────────┐
│  {页面标题}                           {全局筛选}  {操作按钮} │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  {KPI卡片区 - 4-6个卡片横向排列}                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  {主图表区 - 左右分栏或整行}                                │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  {明细表格区}                                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 分析页布局

```
┌────────────┬────────────────────────────────────────────────┐
│  筛选面板  │  页面标题 + 工具栏                             │
│  ────────  ├────────────────────────────────────────────────┤
│            │                                                │
│  □ 时间    │                                                │
│  □ 区域    │             {主图表区}                         │
│  □ 产品    │                                                │
│  □ 指标    │                                                │
│            │                                                │
│  [重置]    ├────────────────────────────────────────────────┤
│  [应用]    │                                                │
│            │             {数据表格区}                       │
│            │                                                │
└────────────┴────────────────────────────────────────────────┘
```

### 大屏布局

```
┌─────────────────────────────────────────────────────────────┐
│                      {大屏标题}                              │
├──────────────┬──────────────────────────┬──────────────────┤
│              │                          │                  │
│  {左侧面板}  │      {中央主区域}        │   {右侧面板}     │
│              │                          │                  │
│  KPI卡片     │      地图/核心图表       │   KPI卡片        │
│  图表        │                          │   图表           │
│              │                          │                  │
│              │                          │                  │
└──────────────┴──────────────────────────┴──────────────────┘
```

## 组件模板

### KPI指标卡

#### 基础样式

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │
│  ↑ 12.3%        │
└──────────────────┘
```

#### 带目标样式

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │
│  目标: 1,500,000 │
│  ████████░░ 85% │
└──────────────────┘
```

#### 对比样式

```
┌──────────────────┐
│  📊 销售额       │
│  本月: ¥128.5万 │
│  上月: ¥114.2万 │
│  ↑ 12.5%        │
└──────────────────┘
```

## 特殊字符参考

### 方框字符

```
单线: ┌ ┐ └ ┘ ─ │ ├ ┤ ┬ ┴ ┼
双线: ╔ ╗ ╚ ╝ ═ ║ ╠ ╣ ╦ ╩ ╬
圆角: ╭ ╮ ╰ ╯
```

### 填充字符

```
░ ▒ ▓ █ ▄ ▀ ■ □ ▣
```

### 符号

```
箭头: ← ↑ → ↓ ↔ ↕ ↖ ↗ ↘ ↙
圆形: ○ ● ◐ ◑ ◒ ◓ ◔ ◕ ◖ ◗
方块: □ ■ ▣ ▤ ▥ ▦ ▧ ▨ ▩
三角: △ ▽ ▷ ◁ ▲ ▼ ▶ ◀
符号: ★ ☆ ♠ ♣ ♥ ♦ ♪ ♫
数字: ① ② ③ ④ ⑤ ⑥ ⑦ ⑧ ⑨ ⑩
表情: 📊 📈 📉 📋 🥧 🍩
```
