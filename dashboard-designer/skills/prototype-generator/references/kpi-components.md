# KPI组件库

本文档定义了BI看板中KPI指标卡的组件类型和设计规范。

## KPI卡片类型

### 1. 基础指标卡

最简单的形式，展示单个指标。

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │
└──────────────────┘
```

**数据需求**:
- 指标名称
- 指标值
- 图标 (可选)

### 2. 趋势指标卡

包含环比/同比变化。

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │
│  ↑ 12.3% 环比   │
└──────────────────┘
```

**数据需求**:
- 指标名称
- 指标值
- 变化值/变化率
- 变化方向 (up/down)

### 3. 目标指标卡

展示目标完成情况。

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │
│  目标: 1,500,000 │
│  ████████░░ 85% │
└──────────────────┘
```

**数据需求**:
- 指标名称
- 实际值
- 目标值
- 完成率

### 4. 对比指标卡

展示与上一周期的对比。

```
┌──────────────────┐
│  📊 销售额       │
│  本月: ¥128.5万 │
│  上月: ¥114.2万 │
│  ↑ 12.5%        │
└──────────────────┘
```

**数据需求**:
- 指标名称
- 本期值
- 对比期值
- 变化率

### 5. 迷你图指标卡

包含小型趋势图。

```
┌──────────────────┐
│  📊 销售额       │
│  ¥ 1,285,000    │   ╱╲
│  ↑ 12.3%        │  ╱  ╲╱╲
└──────────────────┘
```

**数据需求**:
- 指标名称
- 指标值
- 变化率
- 趋势数据数组

## 数值格式化

### 金额格式化

| 原始值 | 格式化后 | 说明 |
|-------|---------|------|
| 1285000 | ¥ 1,285,000 | 标准格式 |
| 1285000 | ¥ 128.5万 | 万单位 |
| 1285000000 | ¥ 12.85亿 | 亿单位 |

### 百分比格式化

| 原始值 | 格式化后 | 说明 |
|-------|---------|------|
| 0.235 | 23.5% | 标准百分比 |
| 12.35 | +12.35% | 带正负号 |
| -5.2 | -5.2% | 负数 |

### 数量格式化

| 原始值 | 格式化后 | 说明 |
|-------|---------|------|
| 3256 | 3,256 | 千分位 |
| 1234567 | 123.5万 | 万单位 |

## 颜色规范

### 趋势颜色

```
上涨/正向: #10b981 (绿色)
下跌/负向: #ef4444 (红色)
持平/无变化: #6b7280 (灰色)
```

### 背景颜色

```
默认背景: #ffffff
警告背景: #fef3c7
成功背景: #d1fae5
危险背景: #fee2e2
```

## 图标推荐

```
📊 销售/统计
💰 金额/收入
📦 订单/产品
👥 用户/客户
📈 增长/趋势
📉 下降
⏱️ 时间/效率
🎯 目标
```

## 布局建议

### 卡片数量

- 桌面端: 4-6个卡片
- 平板端: 2-3个卡片
- 手机端: 1-2个卡片

### 卡片宽度

```css
/* 4个卡片 */
grid-template-columns: repeat(4, 1fr);

/* 6个卡片 */
grid-template-columns: repeat(6, 1fr);
```

### 卡片交互

1. **悬停效果**: 阴影加深或上移
2. **点击效果**: 可跳转到详情页
3. **刷新动画**: 数值更新时的动效
