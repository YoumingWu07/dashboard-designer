# 布局模式参考

本文档定义了BI看板的常用布局模式。

## 布局类型

### 1. 三行布局

最常用的看板布局，适合概览页。

```
┌─────────────────────────────────────────┐
│  头部 (标题 + 工具栏)                    │
├─────────────────────────────────────────┤
│  KPI卡片区 (横向排列)                    │
├─────────────────────────────────────────┤
│  图表区 (1-2行图表)                      │
├─────────────────────────────────────────┤
│  表格区 (数据明细)                       │
└─────────────────────────────────────────┘
```

**CSS实现**:
```css
.container {
    display: flex;
    flex-direction: column;
    gap: 20px;
}
```

### 2. 左右分栏布局

适合分析页，左侧筛选，右侧内容。

```
┌──────────┬──────────────────────────────┐
│          │                              │
│  筛选栏  │        主内容区              │
│  (240px) │                              │
│          │                              │
└──────────┴──────────────────────────────┘
```

**CSS实现**:
```css
.container {
    display: grid;
    grid-template-columns: 240px 1fr;
}
```

### 3. 大屏布局

适合数据大屏展示。

```
┌─────────────────────────────────────────┐
│              大屏标题                    │
├──────────────┬──────────────┬───────────┤
│              │              │           │
│   左侧面板   │   中央区域   │  右侧面板 │
│              │              │           │
│              │              │           │
└──────────────┴──────────────┴───────────┘
```

**CSS实现**:
```css
.container {
    display: grid;
    grid-template-rows: auto 1fr;
    grid-template-columns: 1fr 2fr 1fr;
}
```

### 4. 标签页布局

适合多功能模块切换。

```
┌─────────────────────────────────────────┐
│  [概览] [分析] [明细] [设置]            │
├─────────────────────────────────────────┤
│                                         │
│              当前标签内容               │
│                                         │
└─────────────────────────────────────────┘
```

## 响应式断点

```css
/* 大屏 */
@media (min-width: 1440px) { }

/* 桌面 */
@media (min-width: 1200px) and (max-width: 1439px) { }

/* 平板 */
@media (min-width: 768px) and (max-width: 1199px) { }

/* 手机 */
@media (max-width: 767px) { }
```

## 间距规范

| 元素 | 间距 |
|------|------|
| 页面边距 | 20-24px |
| 组件间距 | 16-20px |
| 卡片内边距 | 16-24px |
| 元素间距 | 8-12px |

## 卡片样式规范

```
圆角: 8-12px
阴影: 0 2px 8px rgba(0,0,0,0.08)
背景: #ffffff
边框: 无 或 1px solid #eee
悬停: 阴影加深 或 上移2px
```
