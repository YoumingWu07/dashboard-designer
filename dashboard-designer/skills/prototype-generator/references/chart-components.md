# 图表组件库

本文档定义了BI看板中常用的图表组件类型和使用场景。

## 图表类型选择

### 趋势类图表

| 图表类型 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 折线图 | 展示指标随时间变化趋势 | 时间维度 + 1个指标 |
| 面积图 | 强调趋势和量级 | 时间维度 + 1个指标 |
| 多折线图 | 对比多个指标趋势 | 时间维度 + 多个指标 |

### 对比类图表

| 图表类型 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 柱状图 | 不同类别的数值对比 | 1个维度 + 1个指标 |
| 条形图 | 类别排名对比 | 1个维度 + 1个指标 |
| 分组柱状图 | 多维度对比 | 1个维度 + 多个指标 |

### 占比类图表

| 图表类型 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 饼图 | 部分与整体关系 | 1个维度 + 1个指标 |
| 环形图 | 饼图变体，更现代 | 1个维度 + 1个指标 |
| 矩形树图 | 多层级占比 | 多层级维度 + 1个指标 |

### 分布类图表

| 图表类型 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 散点图 | 两个变量的关系 | 2个指标 |
| 气泡图 | 三个变量的关系 | 3个指标 |
| 热力图 | 密度分布 | 2个维度 + 1个指标 |

### 地理类图表

| 图表类型 | 适用场景 | 数据要求 |
|---------|---------|---------|
| 区域地图 | 地理区域数据分布 | 区域维度 + 1个指标 |
| 热力地图 | 地理密度分布 | 经纬度或区域 + 指标 |
| 标记地图 | 具体点位标注 | 经纬度 + 信息 |

## 图表配置要素

### 标题和图例

```
- 图表标题: 简洁明了，说明图表内容
- 图例位置: 通常在底部或右侧
- 颜色含义: 使用图例说明不同颜色代表的内容
```

### 坐标轴

```
- X轴: 通常是维度（时间、类别等）
- Y轴: 通常是指标值
- 刻度: 合理的刻度间隔，避免过于密集
- 单位: 在轴标签中标注单位（万、%、件等）
```

### 数据标签

```
- 显示位置: 柱状图在柱顶，折线图在数据点
- 格式化: 数字千分位、保留小数位
- 避免重叠: 数据密集时可隐藏部分标签
```

### 交互功能

```
- 悬停提示: 显示详细数据
- 缩放: 支持时间范围缩放
- 下钻: 点击可查看更详细数据
- 联动: 与其他图表联动筛选
```

## 图表颜色配置

### 配色原则

1. **主色调**: 与整体风格一致
2. **对比色**: 用于强调重点数据
3. **序列色**: 多系列图表使用渐变色系
4. **警示色**: 红色表示下降/异常，绿色表示增长/正常

### 常用配色方案

```
科技蓝: #00d4ff, #0095ff, #6366f1, #8b5cf6
商务蓝: #4f46e5, #06b6d4, #10b981, #f59e0b
素雅色: #64748b, #94a3b8, #cbd5e1, #e2e8f0
```
